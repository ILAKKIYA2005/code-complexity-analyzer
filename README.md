# Code Complexity Analyzer 🚀

A simple web application that analyzes source code and estimates its **time complexity**.  
This tool helps developers and students quickly understand how efficient their code is.

---

## 📌 Features

- Analyze code and estimate **Time Complexity**
- Supports **Python, Java, and C++**
- Detects loop-based complexity automatically
- Displays **visual graph of complexity**
- Clean **Dark Blue UI**
- Simple and beginner-friendly interface

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Chart.js
